# Netflix Exploratory Data Analysis

## Objective

Perform Exploratory Data Analysis (EDA) on the Netflix dataset.

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Analysis Performed

* Dataset overview
* Missing value analysis
* Movies vs TV Shows comparison
* Top 10 Countries
* Release Year distribution

## Outcome

Successfully analyzed the Netflix dataset and visualized important trends using bar charts.
